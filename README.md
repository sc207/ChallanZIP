### Height allocation I'd use

```text
Header + Logo           12%
Company Details          5%   ← fixed 2 lines
Client Box             10-12% ← max 5-6 lines
Products Table         60-65% ← largest area
Summary Box             8-10%
Footer                  5-8%
```

This gives the products the maximum space while still handling:

* Long customer company names
* 2-line addresses
* Client GST
* Client mobile
* Vehicle number
* A5 and A4 printing

The biggest mistake would be letting the company section become 4–5 lines. Keep company info compressed and give that space to the products table.
